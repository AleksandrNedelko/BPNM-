Шаблон BPMN (HTML + CSS)

Что сделано
- Диаграмма с 2 дорожками: «Отдел SEO» и «Отдел SUP».
- BPMN-фигуры: события (start/end), задачи, шлюз (gateway).
- Стрелки: sequence-flow (сплошные) и message-flow (пунктирные) поверх верстки через SVG.

Как открыть
1) Откройте файл `index.html` в браузере.
2) Если диаграмма шире экрана — используйте горизонтальный скролл внутри блока диаграммы.

Как адаптировать под ваш процесс
- Узлы находятся в `index.html` внутри `.lane__grid`.
- Позиция узла задаётся CSS-переменными `--col` и `--row` (сеткой 10×2).
- Стрелки находятся в SVG `.bpmn__flows` в `index.html` (координаты в `viewBox`).

Файлы
- `index.html`
- `styles.css`
