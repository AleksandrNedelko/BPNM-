# BPMN-шаблон (HTML + CSS)

В этой папке лежит **шаблон визуального BPMN‑представления** процесса «постановка задачи SEO → SUP → приём результата».

## Как открыть

- Откройте файл `index.html` в браузере (двойной клик / drag&drop в окно браузера).

## Как редактировать процесс

Диаграмма нарисована в SVG внутри `index.html`.

- **Переименовать шаг**: найдите рядом с блоком `<g class="task">` или `<g class="gateway">` элемент `<text>` и измените содержимое `<tspan>`.
- **Сдвинуть блок**: измените `x`/`y` у `<rect>`, либо поправьте `transform="translate(...)"` у группы.
- **Сдвинуть стрелку/связь**:
  - для прямых линий правьте координаты в атрибуте `d` у `<path class="flow ...">`
  - тип линии:
    - `.flow--seq` — sequence flow (сплошная)
    - `.flow--msg` — message flow (пунктир)

## Роли (как в постановке)

- **LANE: SEO** — постановщик: SEO‑менеджер / SEO‑специалист (отдел SEO)
- **LANE: SUP** — принимающая сторона: SUP / SEO‑менеджер (отдел SUP)

## Что внутри

- `index.html` — разметка страницы + SVG‑диаграмма
- `styles.css` — стили страницы и элементов BPMN
